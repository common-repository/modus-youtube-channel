== Description ==

This nice plugin will display your YouTube Channel, Playlist, or both in responsive rows and columns which you set, you can also modify the amount of thumb nails per page. There is paging added so you can have as many clips as you like.

This plugin is free of charge so if your going to rate it be nice about it;).

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. install using the wordpress install function as with any other plugin, then on any page add the shortcut [youtube-player] with no spaces to include it. Under Settings > MYT Admin you can set your Channel, Playlist or default video clip
